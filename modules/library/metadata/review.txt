org.geotools.factory
--------------------
Basis for a factory implementation, based on java SPI technology forming the core of the
geotools plugin system. These constructs we revised for GeoTools 2.1 to make use of some JAI
and provide a mechanism for "hinting" the lookup process when needed by client applications.

- FactoryFinder file was based on SAXFactory and is credited

 org.geotools.metadata.sql
 -------------------------
 Appears to be low-level support for metadata wrapped over an SQL result set, interested to know
 if this is used. (Answer: not yet used in Geotools. Used in at least one external project however.
 It could be used in a future Geotools version as a more convenient and more detailled way to store
 pre-defined Citations (EPSG, ESRI, etc...) than hard-coded constants.)
