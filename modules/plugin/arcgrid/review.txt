
Module ArcGrid
==============
Module Maintainer, Andrea Amie
IP Review, Jody Garnett July 8th, 2006

org.geotools.gce.arcgrid
------------------------
Plugin providing grid coverage access to arc grid data format. Before GeoTools 2.0 this
was available as a DataSource, updated to a GridCoverageExchange API for GeoTools 2.1.

- No code from pre 2003 remains
