Positioning implementation. Class {@link org.geotools.pt.CoordinatePoint} represents a point in a multi-dimensional space. This space may have an arbitrary number of dimensions. For a two-dimensional space, CoordinatePoint is conceptually equivalent to {@link java.awt.geom.Point2D}.

Class {@link org.geotools.pt.Envelope} represents a box in a multi-dimensional space. For a two-dimensional space, Envelope is conceptually equivalent to {@link java.awt.geom.Rectangle2D}.
