This is the home page for the postgis module.

Dependencies

  • Other Modules: core, defaultcore
  • External Binaries: none
  • Java Versions: J2SE
  • Java Standard Extensions: None

    Overview

    This is not the PostGIS project itself, rather the postgis module provides an implementation of the datasource interface which connects with PostGIS databases.
