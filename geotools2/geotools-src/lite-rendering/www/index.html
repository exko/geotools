Main Features

  • Simple source code (useful for teaching)
  • Fast rendering when the image is displayed for the first time
  • Allows simple but configurable on the fly generalization to speed up rendering of complex features at low scale
  • Low garbage generation during rendering process
  • Low memory consuption (no caching performed)
