{@link java.awt.Shape} implementations used for the rendering of geographic features. Some classes in this package are just ordinary geometric shapes (for example {@link org.geotools.renderer.Arrow2D}, which may be used for rendering wind fields). Some other are more elaborated: they extend {@link org.geotools.renderer.GeoShape}, which has support for setting arbitrary two-dimensional {@linkplain org.geotools.cs.CoordinateSystem coordinate system}.
