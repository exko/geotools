How-To: Build GeoTools2

This is the home page for the gtbuild module.

Dependencies

  • Other Modules: None
  • External Binaries: junit,nbantext,checkstyle,ant-optional,cruisecontrol
  • Java Versions: J2SE, J2ME
  • Java Standard Extensions: None

Overview

gtbuild is not a true module, instead it is the home of the scripts and utilities needed to compile and test all other geotools2 modules.

Main Features

  • build.xml
    This build script is capable of compiling, testing and releasing all (or any subset) of the modules in geotools2. The script does very little of its own work, instead it polls each of the modules in turn and calls their build scripts.
  • Common.xml
    This holds all of the code that most modules will need in order to build.
    Using the ant task in all other modules to call this build file cuts down on the amount of duplicate code that is needed.
