How-To: Build GeoTools2

This is the home page for the gtbuild module.

Dependencies

  • Other Modules: None
  • External Binaries: JTS, log4j, opengis-css
  • Java Versions: J2SE, J2ME
  • Java Standard Extentions: None

Main Features

  • build.xml
    This build script is capable of compiling, testing and releasing all (or any subset) of the modules in geotools2. The script does very little of its own work, instead it polls each of the modules in turn and calles their build scripts.
  • Common.xml
    This holds all of the code that most modules will need in order to build.
    Using the ant task in all other modules to call this build file cuts down on the amount of duplicate code that is needed.
